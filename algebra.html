Квадратные уравнения

Квадратное уравнение имеет вид: ax² + bx + c = 0.

Решение: x = (-b ± √D) / 2a, где D = b² - 4ac.

Теорема Виета

Для уравнения x² + px + q = 0:

  • Сумма корней: x₁ + x₂ = -p.
  • Произведение корней: x₁ · x₂ = q.

Прогрессии

Арифметическая прогрессия:

  • n-й член: aₙ = a₁ + (n - 1)d.
  • Сумма первых n членов: Sₙ = (a₁ + aₙ) · n / 2.

Геометрическая прогрессия:

  • n-й член: bₙ = b₁ · q^(n-1).
  • Сумма первых n членов: Sₙ = b₁ · (qⁿ - 1) / (q - 1).
